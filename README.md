# Filter-Object
I developed a Filter Object web application using HTML, CSS, and JavaScript. This project demonstrates how to filter and display data dynamically on the frontend without using any external libraries or frameworks.
# Feature
Real-time object filtering using JavaScript,
Clean user interface design, 
Dynamic rendering of filtered results, 
Lightweight and fast performance without any frameworks, 
Practical implementation of core JavaScript concepts.

# Technologies
HTML5,
CSS3,
JavaScript.
